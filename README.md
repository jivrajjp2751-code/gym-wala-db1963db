# Vikram Jadhav — Play. Swim. Fuel Your Fitness.

A premium sports destination led by Mr. India 2019 — Vikram Jadhav.

## Features

- **Sports Turf** — Book premium turf slots for Football, Cricket & more.
- **Swimming Pool** — World-class swimming pool with coaching for all ages.
- **Café** — Great food, coffee & refreshments.
- **Gym** — Coming Soon!

## Getting Started

```sh
npm install
npm run dev
```

The dev server will start at [http://localhost:8080](http://localhost:8080).

## Build for Production

```sh
npm run build
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui
